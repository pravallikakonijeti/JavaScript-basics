About Alert Box,Confirm

The alert() method displays an alert box with a specified message and an OK button. An alert box is used for want to make sure information comes from the user

The confirm() method displays a dialog box with a specified message, along with OK and Cancel button. A confirm box is used if we want the user to verify or accept something.
